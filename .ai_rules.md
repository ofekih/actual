# AI Behavioral Rules\n1. NEVER assume approval to begin making code changes.\n2. A question like 'what is next?' or 'how does this work?' is purely informational and NEVER implies a green light to execute the next steps of a plan.\n3. Always wait for EXPLICIT approval (e.g., 'go ahead', 'approved', 'do it') before modifying any source code or triggering irreversible actions.
